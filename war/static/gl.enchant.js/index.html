gl.enchant.jsで触手や多関節を実現するには

var body = new Sprite3D();
var arm = new Sprite3D();

body.addChild(arm);

glでは2DにおけるGroupにあたるものがない。
代わりにSprite3Dに他のSprite3Dを子ノードとして追加できる。

DoGenchaでのDOGA-L3多関節物体(L3C)の実現にはこれをそのまま利用している。
